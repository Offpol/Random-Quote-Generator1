# Random Task Generator

## Автор: [ЕГОР СКЛЯРОВ]
## Описание:
Графическое приложение для генерации случайных задач. Помогает выбрать занятие из предустановленного списка или добавить свои варианты. История сгенерированных задач сохраняется в файл `tasks.json`.

## Технологии:
*   **Язык:** Python 3.x
*   **Библиотеки:** `tkinter` (стандартная), `random`, `json`
*   **Система контроля версий:** Git

## Требования к запуску:
1. Установить Python (версия >= 3.6).
2. Библиотека `tkinter` входит в стандартную поставку Python.
3. Установить Git.

## Инструкция по запуску:
1. Клонируйте репозиторий: `git clone <ссылка_на_репозиторий>`
2. Перейдите в папку проекта.
3. Запустите приложение: `python main.py`

## Возможности приложения:
1.  **Генерация:** Кнопка «Сгенерировать задачу» выбирает случайное занятие из базы.
2.  **История:** Все сгенерированные задачи отображаются в списке справа.
3.  **Фильтр:** Можно отфильтровать историю по типу задачи («Учёба», «Спорт», «Работа»).
4.  **Добавление:** Пользователь может добавить свою задачу с указанием её типа.
5.  **Сохранение:** История и база задач сохраняются в файл `tasks.json` при каждом действии.
