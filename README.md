# Random Quote Generator

## Автор
[Скляров Егор]

## Описание
Приложение "Random Quote Generator" представляет собой графический интерфейс (GUI) на Python, разработанный с использованием библиотеки Tkinter. Оно позволяет генерировать случайные цитаты из предопределенного списка, отображать историю просмотренных цитат, фильтровать их по автору и теме, а также добавлять новые цитаты. История цитат сохраняется в файле `quote_history.json` для последующей загрузки.

## Инструкция по установке и запуску
1. Убедитесь, что у вас установлен Python 3.x.
2. Клонируйте репозиторий: `git clone [URL вашего репозитория]`
3. Перейдите в директорию проекта: `cd random_quote_generator`
4. Запустите приложение: `python main.py`

## Функционал
* **Генерация случайной цитаты**: Кнопка "Сгенерировать цитату" выбирает и отображает случайную цитату из списка.
* **История цитат**: Все сгенерированные цитаты отображаются в виде списка в нижней части окна.
* **Фильтрация**:
    * "Фильтр по автору": Позволяет найти цитаты конкретного автора.
    * "Фильтр по теме": Позволяет найти цитаты по определенной теме.
* **Добавление новых цитат**: Кнопка "Добавить цитату" позволяет ввести новую цитату, автора и тему, которые будут добавлены в список.
* **Сохранение истории**: История цитат автоматически сохраняется в файл `quote_history.json` после каждой генерации и загружается при запуске приложения.

## Примеры использования
1. Запустите приложение.
2. Нажмите "Сгенерировать цитату", чтобы увидеть случайную цитату.
3. Для просмотра истории, прокрутите вниз.
4. Используйте кнопки "Фильтр по автору" и "Фильтр по теме", чтобы найти интересующие цитаты.
5. Нажмите "Добавить цитату", чтобы расширить базу цитат.
